Launch is a clean, efficient, and minimalist Android launcher designed to maximize simplicity, focus, and productivity.

Built for those who value speed, clarity, and control.
Simple. Clean. Powerful.

---

🧠 **Smart App Management**

* Apps sorted automatically by usage frequency
* Rarely used apps grouped alphabetically
* App usage time displayed beside icons
* Weekly phone usage graph (last 7 days)
* Tap any day for detailed breakdown
* Interactive daily pie charts
* Total screen time per day
* One-tap uninstall via long press
* Switch between smooth grid and list view
* Mark apps as favorites
* Toggle between “Favorites Only” and “Show All”
* Organize apps by categories (Social, Productivity, Games, etc.)
* Filter apps by category

---

🔍 **Smart Universal Search**

* Search apps, contacts, and perform web searches
* Built-in calculator in search bar
* Scientific calculations supported
* Quick links to Play Store, YouTube, Maps, and browser
* Tap to call, SMS, or WhatsApp contacts directly
* Voice command support

Example voice commands:

* Call Swaroop
* Message Swaroop
* WhatsApp Swaroop
* Open YouTube
* Search weather tomorrow
* Uninstall WhatsApp
* Bangalore to Mysore (opens Maps route)

* Todo list widget (full task management)

🔐 **Privacy & Permissions**

Launch is 100% open source.
No ads. No tracking.

Permissions used:

* Contacts → Contact search & quick actions
* Phone → Calling support
* SMS → Messaging support
* Storage → Notes, wallpapers, backups
* Usage Stats → App usage tracking & limits (optional)
* Notification Access → Notifications widget (optional)
* Location → Weather widget (optional)
* Microphone → Voice search (optional)
* Camera → Torch feature (optional)

All advanced permissions are optional. Core launcher functionality works independently.
