## Health & Wellness Widgets

### 1. **Water Intake Tracker Widget**
- Track daily water consumption
- Set daily goal (e.g., 8 glasses, 2L)
- Quick increment buttons (+250ml, +500ml, +1L)
- Visual progress bar/circle
- Calendar view for historical data (similar to Physical Activity Widget)
- Reset daily at midnight
- Reminder notifications (optional)
- **Pattern**: Similar to WorkoutWidget with daily tracking and calendar view

### 2. **Sleep Tracker Widget**
- Track sleep duration (manual entry or integration with sleep apps)
- Bedtime and wake time logging
- Sleep quality rating (1-5 stars)
- Weekly sleep average
- Sleep debt calculation
- Calendar view showing sleep patterns
- **Pattern**: Similar to Physical Activity Widget with time-based tracking

### 3. **Mood Tracker Widget**
- Quick mood logging (Happy, Sad, Anxious, Calm, etc.)
- Daily mood entry with optional notes
- Mood trends over time (weekly/monthly graph)
- Calendar view with color-coded moods
- Export mood data
- **Pattern**: Similar to WorkoutWidget with calendar visualization

### 4. **Meditation Timer Widget**
- Set meditation duration
- Timer with ambient sounds (optional)
- Track meditation sessions
- Daily/weekly meditation streak
- Calendar view of meditation history
- **Pattern**: Similar to WorkoutWidget with timer functionality

### 5. **Heart Rate Monitor Widget** (if sensor available)

## Implementation Notes

All widgets should follow these patterns observed in existing code:
- Use `SharedPreferences` for persistence
- Implement enable/disable toggle
- Handle permissions gracefully
- Use `Handler` for periodic updates
- Implement lifecycle methods (`onResume`, `onPause`, `cleanup`)
- Support calendar views for historical data where applicable
- Use consistent UI patterns with existing widgets
- Follow the widget initialization pattern in `WidgetSetupManager`
